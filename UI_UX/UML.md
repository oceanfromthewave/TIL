# UML (Unified Modeling Language)

## UML 이란?

### UML 

- 객체지향 분석/설계 산물(artifacts)을 위한, 표준화된 notation
- Not a method
- Not a development process
- UML은 S/W를 시각화, 명세화, 문서화하기 위한 언어
